# كيف تشغّل الموقع — خطوة بخطوة

## الطريقة الأسهل (بعد تثبيت Node.js)

1. **ثبّت Node.js** (مرة واحدة فقط):
   - افتح المتصفح وادخل إلى: **https://nodejs.org**
   - حمّل النسخة التي تظهر (LTS) وثبّتها.
   - أعد تشغيل الكمبيوتر أو Cursor بعد التثبيت.

2. **شغّل الموقع**:
   - افتح مجلد المشروع:  
     `c:\xampp1\htdocs\Abdalrhman\pro-quant-analytics`
   - **انقر نقراً مزدوجاً** على الملف: **`تشغيل-الموقع.bat`**
   - انتظر حتى تظهر رسالة مثل: **Ready on http://localhost:3000**

3. **افتح الموقع في المتصفح**:
   - افتح Chrome أو أي متصفح.
   - في شريط العنوان اكتب: **http://localhost:3000**
   - اضغط Enter.

بهذا يكون الموقع يعمل على جهازك.

---

## إذا أردت التشغيل من Cursor (الطرفية)

1. من القائمة: **Terminal** ← **New Terminal** (أو اضغط **Ctrl+`**).
2. اكتب الأمر التالي واضغط Enter:
   ```bash
   cd c:\xampp1\htdocs\Abdalrhman\pro-quant-analytics
   ```
3. ثم:
   ```bash
   npm install
   ```
   (انتظر حتى ينتهي التثبيت.)
4. ثم:
   ```bash
   npm run dev
   ```
5. عندما ترى **Ready**, افتح المتصفح على: **http://localhost:3000**

## ملخص

| ماذا تفعل؟              | النتيجة                    |
|-------------------------|----------------------------|
| تثبيت Node.js من الموقع | يصبح أمر `npm` يعمل       |
| تشغيل `تشغيل-الموقع.bat` | يثبت الحزم ويشغّل السيرفر |
| فتح http://localhost:3000 | تظهر صفحة الموقع          |

لإيقاف السيرفر: أغلق نافذة الـ Bat أو في الطرفية اضغط **Ctrl+C**.

## لماذا الموقع لا يشتغل؟ (استكشاف الأخطاء)

### 1. رسالة خطأ `EPERM` أو `spawn EPERM`
معناها: Windows أو برنامج الحماية منع Node.js من تشغيل عملية فرعية.

**ما الذي تفعله:**
- **شغّل من موجه الأوامر العادي (CMD)** وليس من داخل Cursor:
  - اضغط **Win + R**، اكتب **cmd** واضغط Enter.
  - اكتب:
    ```bat
    cd /d c:\xampp1\htdocs\Abdalrhman\pro-quant-analytics
    npm run dev
    ```
- أو انقر نقراً مزدوجاً على **`تشغيل-الموقع.bat`** من مستكشف الملفات (بدون فتح Cursor).
- إذا استمر الخطأ: أضف مجلد المشروع كاستثناء في **Windows Defender** أو برنامج مكافحة الفيروسات (استثناء للمسار الذي فيه المشروع).

### 2. رسالة "Node.js غير مثبت"
- حمّل Node.js من **https://nodejs.org** (LTS) وثبّته.
- أعد تشغيل الجهاز أو على الأقل أغلق وافتح من جديد نافذة CMD أو الملف **تشغيل-الموقع.bat**.

### 3. المنفذ 3000 مستخدم (Port 3000 in use)
- أغلق أي برنامج أو نافذة طرفية أخرى تشغّل موقعاً على المنفذ 3000.
- أو شغّل على منفذ آخر:
  ```bat
  set PORT=3001 && npm run dev
  ```
  ثم افتح في المتصفح: **http://localhost:3001**

### 4. الموقع يفتح لكن الصفحة بيضاء أو لا تُحمّل
- تأكد أنك تفتح **http://localhost:3000** (مع http وبدون s).
- جرّب متصفحاً آخر أو نافذة خاصة (Incognito).
- في الطرفية تأكد أنك ترى **Ready** قبل فتح الرابط.
